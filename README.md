## To Use
First we need to install node.js. So go ahead and do that.

```bash
# Install dependencies
npm install
# Run the app
npm start

# Package the app for mac
npm run package-mac
# Package the app for windows
npm run package-win
# Package the app for linux
npm run package-linux
```
